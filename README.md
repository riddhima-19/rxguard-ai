# RxGuard AI – Drug Interaction & Prescription Safety Checker

## Overview

RxGuard AI is a machine learning-powered healthcare application designed to analyze prescriptions and identify potential medication risks. The system helps improve prescription safety by detecting drug interactions, allergy conflicts, pregnancy-related warnings, and contraindications using healthcare datasets and predictive models.

## Features

* Drug interaction detection
* Prescription risk analysis
* Allergy conflict identification
* Pregnancy safety warnings
* Contraindication screening
* Machine learning-based risk prediction
* Interactive visualizations and analytics dashboard
* Flask-based web interface
